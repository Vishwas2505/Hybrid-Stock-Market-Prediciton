HYBRID INFORMATION MIXING MODULE

The HIMM architecture uses a two-stage feature learning approach. In the first stage, time-series features are extracted from stock prices using a Gated Recurrent Unit (GRU), while semantic features are extracted from text using BERT/Generative AI. In the second stage, both features are combined into a fixed-dimensional mixed feature tensor and processed through dual multilayer perceptrons (MLPs): the Feature-Mixing MLP (token-mixing) and the Interaction-Mixing MLP (channel-mixing).
